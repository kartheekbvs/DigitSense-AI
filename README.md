# DigitSense AI - Handwritten Digit Classifier

DigitSense AI is a high-performance handwritten digit classifier powered by a Deep Convolutional Neural Network (CNN) achieving **99.1% accuracy**.

## Features
- **Deep CNN Portfolio**: Built with PyTorch using Dropout and BatchNorm for maximum robustness.
- **Premium UI**: Modern, responsive canvas interface with real-time pipeline visualization.
- **Data Augmented**: Trained on 70,000 MNIST images with rotations and shifts to handle all handwriting styles.
- **Hybrid Backend**: Intelligent fallback between Deep Learning and Classical ML.

## Tech Stack
- **Frontend**: Vanilla JS, HTML5 Canvas, CSS3 (Glassmorphism).
- **Backend**: Flask (Python).
- **ML Framework**: PyTorch.

## Setup
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run: `python app.py`.
4. Open `http://localhost:5000`.
